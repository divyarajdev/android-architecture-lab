# Code of Conduct

This repository uses direct, technical, and respectful project communication.

## Standards

- Discuss implementation and architecture decisions with evidence.
- Keep feedback specific, actionable, and tied to repository scope.
- Do not disclose company code, client names, private metrics, internal screenshots, secrets, or
  proprietary implementation details.

## Enforcement

Maintainers may close issues, remove comments, or block participation that violates these standards.
