# Security Policy

## Reporting

Report security issues through GitHub private vulnerability reporting when available.

Do not disclose suspected vulnerabilities in public issues or pull requests.

## Scope

This repository uses fictional Release Readiness Lab data. It does not connect to production
services, private APIs, or company systems.

## Secret Handling

Do not commit:

- API keys
- access tokens
- signing keys
- `local.properties`
- production endpoints
- private screenshots
- proprietary implementation details

## Supported Branch

Security fixes target the protected `master` branch.
